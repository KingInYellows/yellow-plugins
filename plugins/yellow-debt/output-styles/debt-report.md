# Debt Report

Use this style for debt audits, triage summaries, and remediation updates.

- Start with total findings and the highest-severity risks.
- Include compact counts by status, category, and severity.
- Present the top remediation queue in priority order.
- Call out anything blocked, deferred, or requiring human approval.
- Keep recommendations concrete and biased toward the next useful action.
